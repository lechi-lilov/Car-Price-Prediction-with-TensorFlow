# Car Price Prediction with TensorFlow

This project uses a Deep Learning approach (TensorFlow/Keras) to predict car prices based on various features.

## 🚀 Project Overview
The goal of this project is to build a regression model that can estimate the price of a vehicle given specific attributes like year, mileage, engine size, and fuel type.

## 🛠️ Technologies Used
* **Python**
* **TensorFlow / Keras** (Neural Networks)
* **Pandas & NumPy** (Data Processing)
* **Matplotlib / Seaborn** (Data Visualization)
* **Scikit-learn** (Data Preprocessing)

## 📊 Dataset
The notebook processes car data, performs exploratory data analysis (EDA), and handles missing values or encoding before feeding it into the neural network.
